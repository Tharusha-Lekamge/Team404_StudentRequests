# Current State ‚Äî Student Request Management System (SRMS)

> Last updated: 2026-04-13  
> Branch: `main`

---

## Overview

A MERN-stack web application that lets students submit academic requests (late add/drop, deadline extensions, repeat-as-first-attempt), staff review and action them, and admins manage accounts. The PHP prototype has been removed; the Node.js/React rewrite is the only codebase here.

## Data Models

### User ([server/models/userModel.js](../server/models/userModel.js))

| Field             | Type   | Notes                                             |
|-------------------|--------|---------------------------------------------------|
| `username`        | String | Required, unique ‚Äî login identifier               |
| `password`        | String | Required, min 8 chars, bcrypt-hashed, `select: false` |
| `passwordConfirm` | String | Validation only; stripped before save             |
| `indexNo`         | String | Student index number                              |
| `accountType`     | String | `student` \| `staff` \| `admin`                   |

Pre-save hook: hashes password with bcrypt, removes `passwordConfirm`.  
Instance method: `correctPassword(candidate, hash)` ‚Äî returns boolean.

| Field               | Type   | Notes                                                      |

| `userName`          | String | Required                                                   |
| `requestType`       | String | `late-add-drop` \| `extend-deadline` \| `repeat-as-first-attempt` |
| `requestInfo`       | String | Required                                                   |

## API Endpoints

### User (`/api/v1/user`)

| Method | Path      | Auth     | Description                  |
|--------|-----------|----------|------------------------------|
| POST   | `/signup` | ‚Äî        | Register student or staff    |
| POST   | `/login`  | ‚Äî        | Authenticate, return JWT     |
| GET    | `/`       | `protect`| Get all users                |
| GET    | `/id`     | `protect`| Get user by index number     |
| PATCH  | `/id`     | `protect`| Update user password         |
| DELETE | `/id`     | `protect`| Delete user by index number  |

### Requests (`/api/v1/requests`)

| Method | Path   | Auth | Description                                      |
|--------|--------|------|--------------------------------------------------|
| POST   | `/`    | ‚Äî    | Create a new request                             |
| GET    | `/`    | ‚Äî    | Get all requests (supports query-string filters) |
| PATCH  | `/`    | ‚Äî    | Set approval status                              |
| GET    | `/get` | ‚Äî    | Get requests by student ID ‚öÝ reads `req.body`   |
| POST   | `/add` | ‚Äî    | Add additional details to a request              |

> No request endpoints are behind `protect` middleware ‚Äî unauthenticated users can read and write all requests.

## Environment Variables Required

Create `server/config.env` (git-ignored ‚Äî never commit this):

```
NODE_ENV=development
port=3000
DB_STRING=mongodb+srv://<user>:<password>@<cluster>.mongodb.net/<dbname>?retryWrites=true&w=majority
JWT_SECRET=<long-random-string>
JWT_EXPIRES=90d
```

| Variable     | Description                                           |
|--------------|-------------------------------------------------------|
| `NODE_ENV`   | `development` enables Morgan logging + detailed errors |
| `port`       | Express listen port                                   |
| `DB_STRING`  | Full MongoDB connection URI                           |
| `JWT_SECRET` | Signing secret for JWTs                               |
| `JWT_EXPIRES`| Token lifetime (e.g. `90d`, `1h`)                    |
